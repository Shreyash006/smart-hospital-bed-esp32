# smart-hospital-bed-esp32
ESP32-based smart hospital bed project using FSR402, OLED display, DHT22, and DC motor.
# Smart Hospital Bed using ESP32

## Features
- Patient presence detection using FSR402
- OLED display output
- DHT22 temperature and humidity monitoring
- Motor activation when patient detected

## Components Used
- ESP32
- FSR402
- DHT22
- OLED Display
- Motor Driver
- 5V DC Motor
